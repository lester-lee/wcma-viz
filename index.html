Explore WCMA

flavor text about artwork connections and exhibits.

You will be presented with five exhibits from which you can choose from to start your exploration. A social network of artworks and exhibitions will be generated based on your choices. Each exhibition is linked to artworks that were shown, and each artwork is linked to exhibitions that it was shown in.

Some exhibits might be empty because of missing data. Apologies!

Please feel free to see their connections, find a random piece you like, or even start over from the beginning to get a new set of exhibits!

Help ‚ùå

Each node (circle) represents either an artwork or an exhibit. The exhibit nodes are pink, while the artwork nodes will have a color that corresponds to its thumbnail!
A connection between two nodes means that an artwork was shown in the connected exhibition.

Interactions
Hover over a node to view details.
Click on a node to select it.
Scroll to zoom in or out.
Drag a node to move it around.
Search WCMA will search the WCMA collection for the currently selected node.
Add Neighbors will look for all nodes that are connected to the currently selected node and add them to the graph.
Load Random Node will randomly select a node and add it to the graph.

Misc.
The data is incomplete, so not every node will have neighbors. Some exhibits might be empty because artwork data is missing. Not every artwork has a thumbnail. Apologies!
